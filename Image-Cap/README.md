## Introduction
Image Captioning.
Module implemention from "[Actor-Critic Sequence Training for Image Captioning](https://arxiv.org/abs/1706.09601)". <br>

## Requirement
* python 3.5
* pytorch 0.3.1
* numpy 1.13.1
* PIL
